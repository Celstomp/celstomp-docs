Follow this guide to create your first cel animation in Celstomp.

## Prerequisites

- Celstomp running (see [Installation](/docs/installation))
- Basic understanding of frame-by-frame animation (helpful but not required)

## Step 1: Launch and Setup

1. **Open Celstomp**: Launch app from web or locally
2. **Dismiss tutorial** (optional): Skip if you want to dive in
3. **Familiarize layout**:
   - **Canvas**: Center drawing area
   - **Timeline**: Bottom frame grid
   - **Tools**: Left toolbar
   - **Layers**: Right panel
   - **Palette**: Color swatches

## Step 2: Create a Simple Shape

We'll start with a bouncing ball - classic first animation.

### Frame 1: Draw Initial Position

1. **Select Brush**: Press `1` or click brush icon
2. **Choose color**: Pick a color from palette (try blue)
3. **Draw circle**: Draw a circle at bottom-left of canvas
   - Hold `Shift` while dragging for straight edge
   - Draw clean circle outline

4. **Add detail**: Optional - add a small shine (white highlight)

### Frame 2: Move Up

1. **Duplicate frame**: Right-click timeline, "Duplicate" or draw directly
2. **Enable onion skin**: Press `O`
3. **Draw new position**: Draw circle higher (upward)
4. **Use reference**: Previous frame ghost shows starting position

### Frame 3: Top of Arc

1. **Duplicate/create frame**: Add next frame
2. **Draw peak**: Circle at highest point
3. **Smooth motion**: Position should look like natural arc

### Frame 4: Move Down

1. **Duplicate/create frame**: Add next frame
2. **Draw descent**: Circle moving down-right
3. **Maintain spacing**: Distance between positions should feel even

### Frame 5: Landing

1. **Duplicate/create frame**: Final frame
2. **Draw ground level**: Circle back at starting height
3. **Complete arc**: Animation forms smooth bounce

## Step 3: Preview Your Animation

1. **First frame**: Press `Q` or `W` to go to frame 1
2. **Play**: Press `Space` to play
3. **Observe**:
   - Motion smoothness
   - Spacing between frames
   - Overall feel

### Adjustments

**Too fast?** Add frames between existing ones (inbetweening)

**Jerky?** Adjust positions for smoother arc

**Not bouncy?** Emphasize top frame with squash/stretch

## Step 4: Refine with Onion Skin

1. **Enable onion skin**: Press `O` (if not on)
2. **Check each frame**: View ghosted previous/next frames
3. **Adjust positions**: Fix any spacing issues
4. **Replay**: Test again

## Step 5: Add Color (Optional)

1. **Create COLOR layer**: Add new layer in layer panel
2. **Select layer**: Click COLOR layer to make active
3. **Fill circle**: Use Fill Brush (`3`) inside circle
4. **Frame-by-frame**: Fill each frame's circle

**Note**: LINE layer is below COLOR layer, so colors show behind lines.

## Step 6: Export Your Animation

1. **File -> Export**: Open export menu
2. **Choose format**: Start with GIF (easy sharing)
3. **Set FPS**: 12 fps (traditional animation feel)
4. **Export**: Click export button
5. **Download**: Save file to computer

## Step 7: Try Variations

Once comfortable, experiment:

### Squash and Stretch

- **Squash**: Flatten circle when hitting ground
- **Stretch**: Elongate circle at peak of arc
- Creates more dynamic, lively motion

### Secondary Action

- Add expression change as ball bounces
- Color shift on impact
- Spark on landing

### Background

- Use **PAPER layer** for static ground line
- Draw once, appears in all frames

## Common First-Time Mistakes

### Too Few Frames

**Issue**: Animation looks jerky

**Fix**: Add more inbetween frames between key poses

### Inconsistent Size

**Issue**: Ball grows/shrinks between frames

**Fix**: Use onion skin (`O`) to reference previous frame, match size

### Wrong Order

**Issue**: Animation plays backward or jumps

**Fix**: Check timeline order, ensure frames progress correctly
