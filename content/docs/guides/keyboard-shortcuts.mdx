Keyboard shortcuts dramatically speed up your animation workflow in Celstomp.

## Why Use Shortcuts?

**Benefits:**
- **Speed**: Draw without clicking toolbar
- **Flow**: Maintain creative focus
- **Efficiency**: Professional workflow habits
- **Precision**: Keyboard is more accurate than mouse for many tasks

## Essential Shortcuts (Memorize First)

Start with these core shortcuts:

| Shortcut | Action | Frequency |
|----------|--------|-----------|
| `?` | Show cheat sheet | High |
| `1`-`8` | Switch tools | Very High |
| `Space` | Play/Pause | Very High |
| `E` / `R` | Next frame | Very High |
| `Q` / `W` | Previous frame | Very High |
| `O` | Toggle onion skin | High |
| `[` / `]` | Brush size | High |
| `Ctrl/Cmd + Z` | Undo | High |

These 8 shortcuts handle 80% of common actions.

## Tool Switching Mastery

Rapid tool switching is key to fast workflow:

### Number Keys 1-8

Memorize tool positions:

| Key | Tool | Mnemonic |
|-----|------|----------|
| `1` | Brush | Start with 1 |
| `2` | Eraser | 2nd tool after brush |
| `3` | Fill Brush | 3 for fill |
| `4` | Fill Eraser | Fill variant |
| `5` | Lasso | Selection |
| `6` | Rect | Rectangle |
| `7` | Eyedropper | Color picker |
| `8` | Clear | End number keys |

### Tool Switching Practice

**Exercise 1: Tool Cycling**
1. Press `1` (Brush), draw a line
2. Press `2` (Eraser), erase part
3. Press `3` (Fill), fill area
4. Repeat without looking at toolbar

**Exercise 2: Context Switching**
1. Draw something with brush (`1`)
2. Pick color with eyedropper (`Alt` + click)
3. Erase mistake (`2`)
4. Fill area (`3`)
5. Clear frame (`8`)

## Frame Navigation Routines

Efficient frame navigation prevents workflow interruption:

### Standard Navigation

**Forward:**
- `E` (most common)
- `R` (alternative)
- Use whichever feels natural

**Backward:**
- `Q` (most common)
- `W` (alternative)

**Jump to ends:**
- `Home`: First frame
- `End`: Last frame

### Navigation Patterns

**Pattern 1: Check work**
`Space` (play) -> `Q` (back) -> `E` (forward) -> `Space` (pause)

**Pattern 2: Edit specific frame**
`Home` -> `E` (count) -> Draw -> `Space` (test)

**Pattern 3: Loop testing**
`Space` (play) -> Watch -> `Space` (pause) -> Edit -> Repeat

## Drawing Shortcuts

### Brush Size Control

Real-time brush size adjustment while drawing:

| Shortcut | Action |
|----------|--------|
| `[` | Decrease size |
| `]` | Increase size |
| `[` + `]` | Large adjustment (press both repeatedly) |

**Practice:**
- Draw stroke, `[` (smaller), draw thinner line
- Draw stroke, `]` (larger), draw thicker line
- Adjust without pausing drawing

### Straight Lines

Draw precise straight lines:

1. **Hold `Shift`**
2. **Click and drag**
3. **Release** - creates perfect straight line
4. **Draw**: Continue for polyline

**Uses:**
- Perspective lines
- Architectural elements
- Precise shapes

### Eyedropper Quick Access

Sample colors without switching tools:

**Method 1: Alt + Click**
1. Draw with brush
2. Hold `Alt`
3. Click canvas to pick color
4. Release `Alt` - resume drawing

**Method 2: Tool 7**
1. Press `7` (eyedropper)
2. Click canvas
3. Press `1` (brush) - new color active

**Speed test:** Alt method is faster for quick color changes.

## Edit Workflow Shortcuts

### Undo/Redo Pattern

Professional animators constantly undo and redo:

**Undo:** `Ctrl/Cmd + Z`
- Correct mistakes instantly
- Try multiple variations
- Experiment freely

**Redo:** `Ctrl/Cmd + Y`
- Restore undone actions
- Cycle through alternatives

**Practice:**
1. Draw something
2. `Ctrl/Cmd + Z` (undo)
3. Draw alternative
4. `Ctrl/Cmd + Z` (undo)
5. `Ctrl/Cmd + Y` (redo original)
6. `Ctrl/Cmd + Y` (redo alternative)

### Clipboard Operations

| Shortcut | Action |
|----------|--------|
| `Ctrl/Cmd + C` | Copy |
| `Ctrl/Cmd + V` | Paste |
| `Ctrl/Cmd + X` | Cut |
| `Ctrl/Cmd + A` | Select all |

**Uses:**
- Copy frame content to another frame
- Reuse elements across frames
- Paste into other software

## Modifier Key Combinations

### Shift + Tool

**Straight lines:** `Shift` + brush drag
**Multi-select:** `Shift` + frame clicks in timeline

### Ctrl/Cmd + Navigation

**Jump:** `Ctrl/Cmd + Home/End` (varies by system)
**Multi-select (alternative):** `Ctrl/Cmd` + frame clicks

### Alt + Tool

**Eyedropper:** `Alt` + brush click
**Pan:** `Alt` + canvas drag

## Common Shortcut Mistakes

### Pressing wrong number

**Issue**: Accidentally press `5` instead of `3`

**Fix**: Muscle memory improves with practice. Start slow, verify before pressing.

### Forgetting to toggle off

**Issue**: Left modifier keys active (holding Shift/Alt)

**Fix**: Tap modifier key to deactivate.

### System conflicts

**Issue**: Browser shortcuts trigger instead (Ctrl/Cmd + T for new tab)

**Fix**: Most conflicts minimal. If major, may need browser-specific workarounds.

## Professional Tips

- **Think in shortcuts**: "I need to go to next frame" -> Press `E`, don't think "click timeline arrow"
- **One-handed operation**: Keep other hand on drawing tablet/mouse
- **Practice drills**: Spend 5 minutes daily doing shortcut exercises
- **Teach friends**: Explaining shortcuts reinforces your learning
- **Create personal patterns**: Develop your own efficient shortcut sequences
