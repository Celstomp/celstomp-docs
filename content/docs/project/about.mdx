## What is Celstomp?

Celstomp is a lightweight, accessible browser-based 2D handdrawn cel animation suite. It's designed to make traditional frame-by-frame animation accessible to everyone without installations or complex setups.

## Project History

### Origins

Celstomp started as a personal project by Ginyoa, who comes from an art background and built the tool while learning to code. The goal was simple: make animation creation more accessible.

### Development Philosophy

**Core principles:**
- **Accessibility**: No downloads, runs in any modern browser
- **Simplicity**: Clean, intuitive interface for artists
- **Traditional workflow**: Frame-by-frame cel animation tools
- **Open source**: Community-driven development and improvement

## Technical Foundation

### Technology Stack

- **HTML5 Canvas**: High-performance drawing surface
- **Vanilla JavaScript**: No frameworks, lightweight and fast
- **CSS**: Modern styling with responsive design
- **Local Storage**: Auto-save and project persistence

### Browser Compatibility

Works in all modern browsers:
- **Chrome**: Full support (recommended)
- **Firefox**: Full support
- **Safari**: Full support
- **Edge**: Full support

## Key Features

### Animation Tools

- **Canvas**: Configurable aspect ratios, zoom/pan navigation
- **Timeline**: Frame-by-frame grid with drag, multi-select
- **Tools**: Brush, eraser, fill, lasso, selection, eyedropper
- **Layers**: LINE, SHADE, COLOR, FILL, PAPER with blend modes
- **Onion skin**: Preview adjacent frames with adjustable colors/opacity

### User Experience

- **Keyboard shortcuts**: Comprehensive shortcut system
- **Auto-save**: Draft persistence in local storage
- **Tutorial**: Built-in guided tour with replay
- **Safety**: Unsaved changes protection

### Export Options

- **MP4 video**: High quality, wide compatibility
- **GIF**: Animated images for web sharing
- **Image sequence**: Individual frames for post-production
- **Project files**: JSON format for continued editing

## Project Goals

### Short-term

- Improve brush system and drawing tools
- Enhance layer blend modes and effects
- Expand export options and formats
- Refine user interface and accessibility

### Long-term

- Advanced timeline features (keyframes, interpolation)
- More export formats (WebM, APNG)
- Cloud storage and project sync
- Collaboration features
- Mobile app versions
