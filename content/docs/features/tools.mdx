Celstomp provides a comprehensive set of drawing tools for traditional frame-by-frame animation.

## Tool Selection

Access tools via keyboard shortcuts 1-8 or click tool buttons in the toolbar:

| Key | Tool | Description |
|-----|------|-------------|
| `1` | Brush | Main drawing tool with pressure support |
| `2` | Eraser | Remove strokes from current layer |
| `3` | Fill Brush | Fill areas with current color |
| `4` | Fill Eraser | Remove filled areas |
| `5` | Lasso | Select custom regions |
| `6` | Rect Select | Select rectangular areas |
| `7` | Eyedropper | Pick colors from canvas |
| `8` | Clear | Clear entire canvas |

## Brush Tool

The brush is Celstomp's primary drawing tool with advanced features.

### Brush Shapes

- **Round**: Standard circular brush
- **Square**: Square-shaped brush
- **Triangle**: Triangle-shaped brush

### Brush Settings

**Size**: Adjust brush size with:
- Keyboard: `[` (smaller), `]` (larger)
- Slider: Use size slider in tool settings
- Pressure: Enable `Use Pressure Size` for pressure-sensitive size

**Opacity**: Control brush transparency
- Range: 0-100% (fully transparent to fully opaque)
- Set via tool settings panel

**Angle**: Rotate brush strokes
- Useful for calligraphy-style effects
- Set angle in tool settings

### Stabilization

Reduce hand tremor with stabilization levels:
- **Off**: No stabilization (most responsive)
- **Low**: Minor smoothing
- **Medium**: Balanced smoothing
- **High**: Maximum smoothing (least responsive)

### Pressure Sensitivity

- **Size**: Brush size responds to pen/tablet pressure
- **Opacity**: Stroke opacity responds to pressure
- **Tilt**: Brush angle responds to pen tilt (supported devices)

## Eraser Tool

Remove content from the current layer.

### Eraser Options

- **Soft edges**: Blended eraser for smooth transitions
- **Hard edges**: Sharp eraser for precise removal
- **Size**: Adjustable like brush size

**Right-click** eraser button for quick options menu.

## Fill Tools

### Fill Brush

Flood fill areas with current color:
- **Tolerance**: Adjust color matching sensitivity
- **Anti-alias**: Smooth fill edges (recommended)

### Fill Eraser

Remove filled areas while preserving stroke work:
- Useful for correcting fills without redrawing lines
- Selectable via context menu (right-click tool button)

## Selection Tools

### Lasso Selection

Select custom, non-rectangular areas:
- Draw outline around desired area
- Release to complete selection
- Move or modify selected content

### Rectangular Selection

Select rectangular regions:
- Click and drag to create selection
- Perfect for precise area operations
- Use with lasso for complex selections

## Eyedropper

Pick colors from the canvas:
1. Select eyedropper tool or hold `Alt` with brush
2. Click on canvas to sample color
3. Selected color becomes active in palette

## Tool Settings Panel

Access detailed settings for each tool via the folded panels:

- **Brushes Panel**: Shape, size, opacity, angle
- **Settings Panel**: Stabilization, pressure options
- Click panel headers to expand/collapse

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `1`-`8` | Select tools 1-8 |
| `[` / `]` | Decrease/increase brush size |
| `Shift` + drag | Draw straight line |
| `Alt` | Temporarily use eyedropper |
| `Delete` | Clear current frame/selection |
| `Ctrl/Cmd + Z` | Undo stroke |
| `Ctrl/Cmd + Y` | Redo stroke |

## Tips

- Use **stabilization** for cleaner lines, especially with mouse
- Enable **pressure sensitivity** when using a drawing tablet
- **Straight line mode** (`Shift` + drag) for precise lines
- **Eyedropper shortcut** (`Alt`) for quick color sampling without switching tools
- Adjust **tolerance** for fill tools when working with complex color areas
