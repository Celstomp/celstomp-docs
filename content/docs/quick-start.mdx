Get up and running with Celstomp in just a few minutes.

## Using the Web Version

The quickest way to try Celstomp is to visit the live site:

1. Go to [ginyo.space/celstomp/](https://ginyo.space/celstomp/)
2. Click the **?** button in the app to see all keyboard shortcuts
3. Start drawing! The built-in tutorial will guide you through the basics

## Running Locally

### Option 1: Using Node.js (Recommended)

```bash
# Clone the repository
git clone https://github.com/Celstomp/celstomp.git
cd celstomp

# Install dependencies
npm install

# Start the development server
npm run dev
```

The dev server will start at `http://localhost:5173`

### Option 2: Using Python (No Node.js required)

**Linux / Mac (Terminal):**
```bash
./run-dev.command
```

**Windows:**
Double-click `run-dev.bat` or run from command prompt:
```cmd
run-dev.bat
```

The Python server will start at `http://localhost:8000`

## Your First Animation

1. **Select a tool** - Use keys 1-8 or click the tool buttons
2. **Draw on the canvas** - The default canvas is 16:9 aspect ratio
3. **Add a frame** - Press `E` to move to the next frame or click the timeline
4. **Toggle onion skin** - Press `O` to see your previous frame (ghosted)
5. **Play back** - Press `Space` to preview your animation
6. **Export** - Export as GIF, MP4, or image sequence

## Keyboard Shortcuts Basics

| Key | Action |
|-----|---------|
| `?` | Show all shortcuts |
| `1-8` | Select tools |
| `Space` | Play/Pause |
| `E` | Next frame |
| `Q` / `W` | Previous frame |
| `O` | Toggle onion skin |
| `[` / `]` | Adjust brush size |

Press `?` in the app to see the full list of shortcuts.
